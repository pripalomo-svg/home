Como usar a Central de Organização

Guia passo a passo · não precisa ser programadora

🏠 Painel principal 📈 Investimentos ✏️ Cadastrar pacientes
Jeito mais fácil no Windows: dê duplo-clique em ATUALIZAR.bat na pasta organizacao. Ele importa tudo e atualiza os painéis automaticamente.
PASSO 1

👀 Ver como está ficando (só olhar)

Abra estas pastas no seu computador e dê duplo-clique nos arquivos HTML:

ArquivoO que mostra
organizacao/index.htmlTudo: família, pacientes, finanças, agenda
organizacao/investimentos.htmlSua carteira + projeção 20 anos
organizacao/cadastro_pacientes.htmlFormulário para preencher pacientes

Funciona offline, sem internet. Se abrir no Chrome/Edge, está certo.

PASSO 2

✏️ Adicionar ou mudar informação

Os dados ficam em arquivos .csv (planilhas simples) na pasta organizacao/templates/.

Abra com Excel, Google Sheets ou Bloco de Notas:

Quer mudar…Edite este arquivo
Investimentos (Itaú, XPML11, Tesouro…)templates/investimentos.csv
Pacientes do consultóriotemplates/pacientes.csv ou use o formulário visual
Finanças (receitas/despesas)templates/financas.csv
Agenda / compromissostemplates/agenda.csv
YouTubetemplates/youtube_videos.csv

Exemplo — adicionar um CDB no Itaú: abra investimentos.csv, vá na última linha e adicione:

CDB Itaú 120%;cdb;Itaú;;;5000;5000;;;12.5;;2026-07-14;100;#ec7000;CDB 2 anos

Salve o arquivo. As colunas são separadas por ; (ponto e vírgula).

PASSO 3

🔄 Atualizar o painel (depois de editar)

Depois de salvar o CSV, rode estes comandos no terminal.

No Windows: abra o PowerShell ou Terminal, vá até a pasta:

cd C:\Users\pripa\OneDrive\...\organizacao

(ou o caminho onde está a pasta organizacao no seu PC)

Para investimentos:

python importar_investimentos.py templates/investimentos.csv
python gerar_investimentos.py

Para o painel geral (pacientes, finanças, agenda…):

python importar_dados.py todos
python gerar_dashboard.py

Para tudo de uma vez (investimentos + geral):

python importar_investimentos.py templates/investimentos.csv
python importar_dados.py todos
python gerar_investimentos.py
python gerar_dashboard.py

Depois: volte ao PASSO 1 e abra o HTML de novo (F5 para atualizar se já estiver aberto).

PASSO 4

🧠 Pacientes — jeito mais fácil (sem CSV)

  1. Abra cadastro_pacientes.html
  2. Preencha nome, telefone, queixa de cada paciente
  3. Clique Exportar CSV
  4. Salve como templates/pacientes.csv
  5. Rode: python importar_dados.py pacientes templates/pacientes.csv
  6. Rode: python gerar_dashboard.py
PASSO 5

🔗 Sincronizar com o Notion (automático)

Seus atendimentos e prontuários do Notion podem entrar sozinhos no painel.

  1. Uma vez só: siga o guia notion/COMO_CONFIGURAR_API.md (criar integração + salvar token em notion/token.txt)
  2. Para sincronizar agora: duplo-clique em SINCRONIZAR_NOTION.bat
  3. Para todo dia: Agendador de Tarefas do Windows → agende SINCRONIZAR_NOTION.bat (ex.: 07:00)
  4. Abra index.html — sessões e resumos clínicos já estarão lá

Sem API? Exporte do Notion como ZIP, salve em notion/Export.zip e rode o mesmo SINCRONIZAR_NOTION.bat.

PASSO 6

📈 Investimentos — testar projeção

  1. Abra investimentos.html
  2. Mova o slider Aporte mensal (ex.: R$ 500, R$ 1.000)
  3. Veja quanto terá em 5, 10 e 20 anos — atualiza na hora, sem rodar comando
  4. Para mudar valores dos ativos: edite o CSV (Passo 2) e regenere (Passo 3)
Problemas comuns
  • "python não encontrado" → instale Python em python.org, marque "Add to PATH"
  • Planilha bagunçou as colunas → use ponto e vírgula (;) como separador no Excel ao salvar CSV
  • Painel não mudou → rodou o comando gerar_...? Deu F5 no navegador?
  • Primeira vez → rode python organizacao.py init uma vez só
